undersample.py	This is a module that wraps the different undersampling techniques.
undersample-script.py	This is a script that will output a new X and y after undersampling.

oversample.py	This is a module that wraps the different oversampling techniques.
oversample-script.py	This is a script that will output a new X and y after oversampling.

undersample-test.py	This is a script that will train and cross
validate a model using undersampling. It does cross-validation on 20% of
the original data.

oversample-test.py	This is a script that will train and cross
validate a model using oversampling. It does cross-validation on 20% of
the original data.
